# Staerium Python Project

Starter Python workspace for Staerium experiments and prototypes.

## Requirements
- Python 3.12+
- pip (comes with recent Python versions)

## Getting Started
1. Create a virtual environment (recommended): `python -m venv .venv`
2. Activate it:
   - Unix/macOS: `source .venv/bin/activate`
   - Windows: `.venv\\Scripts\\activate`
3. Install tooling and dependencies: `pip install -r requirements.txt`
4. Run the app: `python src/main.py`

## Testing
- Execute `pytest` to run the unit test suite in `tests/`.

## Project Layout
- `src/` — Application source code (`__init__.py`, `main.py`)
- `tests/` — Pytest-based unit tests
- `requirements.txt` — Runtime and tooling dependencies
- `.gitignore` — Ignores virtual environments, caches, and editor files

Update these files as the project evolves.
